# 🔍 한글 맞춤법 검사기 Chrome 확장 프로그램

![Version](https://img.shields.io/badge/version-1.1.1-blue)
![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-green)
![License](https://img.shields.io/badge/license-MIT-yellow)

웹페이지의 한글 텍스트에서 맞춤법 오류를 자동으로 찾아 빨간색으로 표시해주는 Chrome 확장 프로그램입니다.

**두 가지 검사 방식 지원:**
- 🔧 **룰 베이스**: 빠르고 간단한 기본 검사 (서버 불필요)
- 🚀 **API 기반**: hanspell 라이브러리를 사용한 정확한 검사 (서버 필요)

## ✨ 주요 기능

- 🚀 **자동 맞춤법 검사**: 페이지 로드 시 자동으로 맞춤법 검사 실행 (기본 활성화)
- ⚡ **실시간 검사**: 입력창에 타이핑할 때마다 자동으로 맞춤법 검사 (1초 딜레이)
- 📝 **수동 맞춤법 검사**: 버튼 클릭으로 언제든지 검사 실행
- 🔴 **시각적 표시**: 오타를 빨간색 밑줄로 표시하여 쉽게 식별
- 🖱️ **클릭하여 수정**: 오류를 클릭하면 올바른 표현으로 즉시 수정 가능
- 💡 **도움말 표시**: 오류 위에 마우스를 올리면 올바른 표현을 툴팁으로 표시
- 📊 **통계 정보**: 검사한 단어 수와 발견된 오류 개수 표시
- ⚙️ **자동 검사 토글**: 페이지 로드 시 자동 검사 기능을 켜고 끌 수 있음

## 🚀 빠른 시작

### 방법 1: 기본 설치 (룰 베이스)

1. Chrome 브라우저에서 `chrome://extensions/` 접속

2. 우측 상단의 **개발자 모드** 활성화

3. **압축해제된 확장 프로그램을 로드합니다** 클릭

4. 이 프로젝트 폴더 선택

5. 완료! 바로 사용 가능합니다.

### 방법 2: API 기반 설치 (더 정확한 검사)

**상세한 설치 가이드는 [SETUP_GUIDE.md](SETUP_GUIDE.md)를 참조하세요.**

간단 요약:
```bash
cd server
npm install
npm start
```

서버 실행 후 `content.js`에서 `USE_API = true`로 설정

### ⚙️ 코드를 수정한 경우

코드를 수정한 후에는 반드시 확장 프로그램을 **새로고침**해야 합니다:
1. `chrome://extensions/` 페이지로 이동
2. "한글 맞춤법 검사기" 확장 프로그램의 새로고침 버튼(🔄) 클릭
3. 테스트하려는 웹페이지도 새로고침

## 📖 사용 방법

### 자동 검사 모드 (기본)

1. 확장 프로그램을 설치하면 **자동으로 모든 웹페이지에서 맞춤법 검사가 실행됩니다**.

2. 페이지를 열면 즉시 오타가 빨간색으로 표시됩니다.

3. **입력창(input, textarea)에 타이핑하면 자동으로 실시간 맞춤법 검사**가 실행됩니다.
   - 타이핑을 멈춘 후 1초 뒤에 자동으로 검사
   - 입력한 텍스트의 오타가 즉시 빨간색으로 표시됨

4. 오류 위에 마우스를 올리면 올바른 표현을 확인할 수 있습니다.

5. 오류를 클릭하면 올바른 표현으로 수정할 수 있습니다.

### 수동 검사 모드

1. 확장 프로그램 아이콘을 클릭합니다.

2. **"페이지 로드 시 자동 검사"** 체크박스를 해제하면 자동 검사가 비활성화됩니다.

3. 이제 **"맞춤법 검사 시작"** 버튼을 클릭해야만 검사가 실행됩니다.

4. **"표시 지우기"** 버튼을 클릭하면 모든 하이라이트가 제거됩니다.

### 기타 기능

- 🔄 **자동 검사 토글**: 팝업에서 자동 검사를 켜거나 끌 수 있습니다 (설정은 저장됨)
- 🗑️ **표시 지우기**: 하이라이트된 오류 표시를 모두 제거합니다
- 📊 **통계 확인**: 팝업에서 발견된 오류 개수와 검사한 단어 수를 확인할 수 있습니다

## 🐛 문제 해결

### "Extension context invalidated" 오류

이 오류는 확장 프로그램을 업데이트하거나 새로고침한 후 이미 열려있던 페이지에서 발생합니다.

**해결 방법:**
1. 오류가 발생한 페이지를 **새로고침** (F5 또는 Cmd/Ctrl+R)
2. 확장 프로그램이 자동으로 다시 로드됩니다

**참고:** 
- 이 오류는 정상적인 동작이며, 페이지 새로고침으로 해결됩니다
- 확장 프로그램은 캐시된 설정을 사용하여 계속 작동을 시도합니다

### 표시가 나타나지 않는 경우

1. **개발자 도구 콘솔 확인**:
   - F12를 눌러 개발자 도구 열기
   - Console 탭에서 "한글 맞춤법 검사기" 관련 로그 확인
   - 오류 메시지가 있는지 확인

2. **확장 프로그램 새로고침**:
   - `chrome://extensions/` 페이지로 이동
   - 확장 프로그램 새로고침 버튼(🔄) 클릭
   - 테스트 페이지도 새로고침

3. **테스트 페이지 사용**:
   - 프로젝트 폴더의 `test.html` 파일을 브라우저에서 열기
   - 의도적으로 오타가 포함된 테스트 문장들이 있음
   - 여기서 정상 작동하는지 확인

4. **콘솔 로그 확인**:
   - 맞춤법 검사 실행 시 콘솔에 다음과 같은 메시지가 표시됨:
     - "한글 맞춤법 검사기 Content Script 로드됨"
     - "오타 발견: ..." 
     - "하이라이트 생성: ..."
   - 이 메시지들이 없다면 content script가 로드되지 않은 것

5. **페이지 권한 확인**:
   - 일부 웹사이트는 보안상 확장 프로그램 실행을 차단할 수 있음
   - 다른 웹사이트에서 테스트해보기
