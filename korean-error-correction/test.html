🔍 한글 맞춤법 검사기 테스트 페이지

이 페이지는 한글 맞춤법 검사기 Chrome 확장 프로그램을 테스트하기 위한 페이지입니다.

아래 문장들에는 의도적으로 맞춤법 오류가 포함되어 있습니다. 확장 프로그램을 실행하여 오류를 찾아보세요!

📖 사용 방법

  1. 자동 모드: 페이지를 열면 자동으로 오류가 표시됩니다.
  2. 실시간 검사: 아래 입력창에 타이핑하면 자동으로 검사됩니다.
  3. 오류 위에 마우스를 올리면 올바른 표현을 볼 수 있습니다.
  4. 오류를 클릭하면 수정할 수 있습니다.

⚡ 실시간 맞춤법 검사 테스트

아래 입력창에 한글을 입력하면 1초 후 자동으로 맞춤법 검사가 실행됩니다.

💡 팁: 타이핑을 멈추면 1초 후 자동으로 오타가 빨간색으로 표시됩니다!

테스트 문장 1: 부정 표현

오늘은 날씨가 좋지 않아서 밖에 나가지 않됩니다. 집에 있는 것이 더 좋을 것 같아요. 약속을 취소해도 않될까요?

테스트 문장 2: '되다'와 '돼' 혼동

이 일이 잘 되요? 저는 되게 기대되요. 빨리 되가지고 싶어요. 언제쯤 대요?

테스트 문장 3: '웬'과 '왠' 혼동

웬지 오늘은 기분이 좋네요. 웬만하면 일찍 집에 가려고 해요. 그런데 왠일인지 일이 많아요.

테스트 문장 4: 자주 틀리는 표현

어떻해야 할지 모르겠어요. 몇일 후에 다시 연락드릴게요. 바램이 이루어졌으면 좋겠어요.

📝 예시 문장 모음

  • 금새 시간이 지나갔네요.
  • 방이 너무 넒어요.
  • 문이 닫쳐있어요.
  • 김치를 담굼하고 있어요.
  • 있따 만나요!

테스트 문장 5: 복합 오류

오늘 회의는 몇시에 시작되요? 웬지 않돼는 것 같은데, 어떻해야 할까요? 몇일 기다려봐야 할 것 같아요. 바램은 금새 이루어지지 않아요.

테스트 문장 6: 긴 문단

프로젝트가 잘 진행되요. 팀원들과 협력이 잘 돼서 다행이에요. 웬지 이번에는 성공할 것 같은 기분이 드네요. 어떻해야 더 효율적으로 일할 수 있을까 고민했는데, 웬만하면 매일 회의를 하는 것이 좋을 것 같아요. 몇일 후면 결과를 볼 수 있을 거예요. 우리의 바램이 금새 이루어지길 기대합니다!

올바른 문장 (오류 없음)

이 문장은 맞춤법이 모두 올바릅니다. 안녕하세요. 날씨가 좋네요. 오늘은 화창한 날입니다. 내일 만나요. 감사합니다. 잘 부탁드립니다.

💡 팁: "표시 지우기" 버튼을 클릭하면 모든 하이라이트가 제거됩니다.

이 확장 프로그램은 기본적인 맞춤법 오류를 감지합니다.
